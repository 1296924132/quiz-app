# TREA 项目概述

项目根目录：`D:\TRAE文件`（也是 Git 根目录）

## 模块划分

| 子目录 | 说明 |
|--------|------|
| `申报书生成/` | 暑期社会实践院级项目申报书（附件3）Word 生成 |
| `刷题神器/` | 公文写作刷题离线前端（纯 HTML） |
| `反恐实训作业/` | 反恐怖现场处置实训期末作业 |
| `暑假实习意向分析/` | 暑期实习意向汇总分析（Excel + Word 报告） |
| `PPT/` | 幻灯片生成脚本 |
| `资料/` | 原始素材（学习通题库 docx 等） |

## 技术栈

- Python 3.6+，主要依赖 `python-docx`、`openpyxl`、`pandas`
- 前端为纯 HTML/CSS/JS（离线可用）

## 工作目录

**必须为 `D:\TRAE文件`**，不要设为子目录。

## 可用 Skill

- `/generate-form` — 生成申报书
- `/docx-template` — Word 模板填充
- `/ppt-master` — PPT 生成与美化
- `/ralph-loop` — 自治开发循环
